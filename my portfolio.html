Tic tac toe game (2025-2026)
This project is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript. It allows two players to take turns marking "X" and "O" in a 3x3 grid, with the game checking for a winner after each move. The interface is clean and responsive, with a reset button to start a new game and a winner announcement once someone wins. The game also includes a disable feature to prevent further moves once the game is over. This project demonstrates the ability to manage user interactions, game logic, and state changes in a browser environment.
Rock paper Scissor game (2025-2026)
This is a simple Rock Paper Scissors game built with HTML, CSS, and JavaScript. The game allows the user to choose between rock, paper, or scissors, and the computer randomly selects one of the three options as well. The score is tracked for both the user and the computer, with visual feedback provided for each round, such as winning, losing, or a draw. The game is interactive with hover effects and a clean user interface. This project demonstrates the use of basic web technologies to create an engaging game experience..
